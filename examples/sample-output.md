# Sample Post-Publish Review Output

## Input

```text
Checkpoint: 24h
Title: 收藏了 20 个 AI 工具，为什么还是发不出来？
Cover: 你缺的不是工具，是内容流程
Data: low click, medium read, high save among readers
Goal: saves and follows
```

## Output Shape

Return the likely bottleneck, whether title/cover/opening/body should be changed, what signal is worth keeping, and one next experiment.

## Boundary Note

This sample uses user-provided material and public-safe judgment only. It does
not require platform login, automatic publishing, private analytics connectors,
or copying another creator's content.
